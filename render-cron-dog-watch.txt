# Render Blueprint — Dog Watch collector runs as a Cron Job, separate from the web service.
#
# In the Render dashboard, create a Cron Job with:
#   Schedule:  0 6 * * *          (daily at 6:00 AM UTC)
#   Command:   python manage.py dog_watch_sync
#
# Each run:
#   1. Downloads the USDA active licensees spreadsheet (adds new sites only)
#   2. Checks APHIS for new inspection reports on facilities due for refresh
#   3. Parses violations only for newly discovered reports (never re-processes old ones)
#
# Already-processed reports (violations_parsed=True) are never touched.
# Backfill historical reports manually:
#   python manage.py dog_watch_sync --parse-violations-only
#
# First-time setup (run once via Render Shell to backfill everything):
#   python manage.py dog_watch_sync --force
#
# If a run gets stuck:
#   python manage.py dog_watch_sync --clear-lock
#
# Clear lock and immediately start a new run:
#   python manage.py dog_watch_sync --clear-lock --force
#
# To skip the USDA import step (not recommended for cron):
#   python manage.py dog_watch_sync --skip-usda-import
