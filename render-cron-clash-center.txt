# Render Blueprint — Clash Center collector runs as a Cron Job, separate from the web service.
#
# In the Render dashboard, create a Cron Job with:
#   Schedule:  0 * * * *           (every hour, on the hour)
#   Command:   python manage.py clash_center_sync
#
# Required env vars (same as web service):
#   CLASH_ROYALE_API_TOKEN=...          — API key from developer.clashroyale.com
#   CLASH_CENTER_PLAYER_TAG=#YOURTAG    — your Clash Royale player tag
#
# API requests go through https://proxy.royaleapi.dev/v1 (hardcoded).
# Whitelist 45.79.218.79 on your API key at developer.clashroyale.com (proxy IP).
#
# Optional env vars:
#   CLASH_CENTER_SYNC_INTERVAL_HOURS=1
#   CLASH_CENTER_MAX_BATTLES_PER_SYNC=2500
#   CLASH_CENTER_API_DELAY_SECONDS=0.4
#
# First-time backfill (run once via Render Shell):
#   python manage.py clash_center_sync --force
#
# Refresh card catalog only:
#   python manage.py clash_center_sync --cards-only
#
# If a run gets stuck:
#   python manage.py clash_center_sync --clear-lock --force
#
# The web UI at /games/clash-center/ reads from the database only — no API calls.
